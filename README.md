## 使用方法

1. 在 `main.py` 同级目录下创建 `src`、`dst`、`log` 三个文件夹。

2. 在 `src` 文件夹中放入自己想要压缩的 bitmap 图片（`*.bmp`），随后在 `main.py` 同级目录下打开命令行，输入

```shell
python main.py
```

3. 等待其跑完即可
